# For developers

## Set up

```
$ git clone git@github.com:sanographix/kitekure.git kitekure
$ cd kitekure
$ npm install
```

## Start local server

```
$ npm start
```

Default URL is <http://localhost:3000/>.

## Build

```
$ npm run build
```

Assets will output to `build/`.
